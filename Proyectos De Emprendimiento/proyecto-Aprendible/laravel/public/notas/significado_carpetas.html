Carpetas.

APP: en esta carpeta se encuentra el codigo fuente de nuestra aplicacion, y todas las clases que nosotros creemos, se alojaran en este directorio.

Bootstrap: en esta carpeta estara alojado el archivo app.php que es el encargado de inicializar el framwork de laravel.

Subcarpeta Cache: esta carpeta es la encargada de guardar los archivos temporales de cache de laravel para poder ahorrar tiempo en algunas operaciones y optimizar el rendimiento de la aplicacion.

config:esta carpeta es la encargada de tener todos los archivos de configuracion de nuestra aplicacion.

database:aqui se guardaran todo tipo de informacion, desde migraciones, seeds(semilleros)

public:aqui estaran los archivos publicos, uno que destaca es el llamado index.phpque funcionara como punto de entrada de todos las peticiones que le haremos a nuestra pagina web.

vendor:Aqui se guardan las dependencias de laravel.
